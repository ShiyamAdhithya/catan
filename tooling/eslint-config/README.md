# @catan/eslint-config

Shared ESLint configurations for the Catan workspace.

## Configurations

### base.js

Repository-wide linting rules.

### library.js

Configuration for reusable workspace packages.

### react.js

Configuration for React applications.

## Design Principles

- Flat Config
- TypeScript-first
- Prettier owns formatting
- Minimal custom rules
- Prefer upstream recommended configurations
